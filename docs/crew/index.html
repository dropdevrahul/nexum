Install & run

One binary. No backend required.

# build
cd tui && cargo build --release

# run from anywhere inside a git repo
./target/release/crew

# вҖҰor launch headless from the shell (creates a worktree, runs the harness)
./target/release/crew --new "add billing endpoint" --harness cursor

# in the dashboard: [n] launch В· [enter] chat В· [P] push+PR В· [?] keymap

Standalone by default. crew creates worktrees natively and keeps its own registry under .crew/ вҖ” it needs only git and whichever agent CLI you launch. Drop in the optional nexum engine and it also surfaces observed sessions and headless delegation.
